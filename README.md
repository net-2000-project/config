# NET-2000: Децентрализованная Резидентная VPN-Сеть

Проект высокопроизводительной VPN-инфраструктуры, сочетающей мощь серверных узлов (Datacenter) и скрытность домашних прокси (Residential). 

## 🛡 Философия проекта
Вместо аренды публичных прокси, мы строим сеть на базе микрокомпьютеров (Raspberry Pi, MacBook), размещенных у доверенных лиц в Европе и СНГ. Это делает трафик технически неотличимым от обычного домашнего серфинга и защищает от ковровых блокировок IP-адресов дата-центров.

## 🏗 Архитектура "Master-Node"
- **Центральный хаб (Aeza, RU)**: Точка входа для всех пользователей. Обрабатывает авторизацию и первичную маскировку трафика (VLESS + Reality).
- **Резидентные шлюзы (Reverse Tunnels)**: Удаленные узлы (Венгрия, Сербия, Грузия) подключаются к Хабу через обратные туннели. Весь целевой трафик (Instagram, YouTube, Google) "выходит" в интернет именно через них.
- **DPI Bypass**: Использование gRPC-транспорта для стабильной работы в мобильных сетях РФ.

## 🚦 Текущий статус
- **Core (Aeza)**: 🟢 Активен. Поддержка TCP и gRPC.
- **Residential Exit (HU)**: 🟡 Тестирование перехода на Reverse Tunnel.
- **Config Sync**: 🟢 Автоматическое обновление через GitHub.
